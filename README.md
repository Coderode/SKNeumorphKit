## Installation

SKNeumorphKit is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'SKNeumorphKit'
```
### Podfile Installation

```
use_frameworks!

platform :ios, '13.0'

target 'SKNeumorphKit_Example' do
  pod 'SKNeumorphKit', :path => '../'
end

```
